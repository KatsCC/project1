HavePLANs

HavePLANs 는 사람들 끼리 약속을 만들어 공유하는 페이지입니다.

장소, 일시, 내용 등을 입력하여 새로운 약속을 생성하고, 댓글과 참여버튼을 통해 의사표현이 가능하므로 편리하게 약속을 만들기 위함을 생각하며 만들었습니다.
